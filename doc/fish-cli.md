fish-cli
---------
### Usage

    fish-cli -v 或 fish-cli --version                查看版本
    fish-cli -h 或 fish-cli --help 或 错误指令       查看帮助信息
    fish-cli -c 或 fish-cli --colorword              滚动彩色字母
    fish-cli -d 或 fish-cli --day                    查看万年历
    fish-cli -m 或 fish-cli --mail                   发送邮件
    fish-cli --weather=城市                          查看指定城市天气预报
